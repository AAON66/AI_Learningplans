# 管理员看板功能说明

## 功能概述

管理员看板提供了系统的全局数据视图,帮助管理员快速了解系统运营状况。

1. 登录管理员账号: http://localhost:5176/admin/login

   - 密码: 250378

2. 登录后自动进入看板页面: http://localhost:5176/admin

## 看板功能

### 1. 核心指标卡片 (4个)

#### 总用户数
- 显示系统注册用户总数
- 显示最近7天新增用户数
- 图标: 用户群组图标
- 颜色: 蓝色

#### VIP会员
- 显示当前有效VIP会员数
- 显示VIP会员占比
- 图标: 星星图标
- 颜色: 琥珀色

#### 学习计划
- 显示创建的学习计划总数
- 显示最近7天新增计划数
- 图标: 剪贴板图标
- 颜色: 绿色

#### 完成率
- 显示学习计划完成率
- 显示已完成计划数量
- 图标: 勾选图标
- 颜色: 紫色

### 2. 计划状态分布

可视化展示不同状态的学习计划分布:
- 进行中 (蓝色进度条)
- 已完成 (绿色进度条)
- 已暂停 (琥珀色进度条)
- 草稿 (灰色进度条)

每个状态显示:
- 数量
- 占比(进度条长度)

### 3. VIP卡密统计

显示VIP卡密的使用情况:
- 总卡密数
- 已使用数量 (绿色)
- 未使用数量 (蓝色)
- 使用率百分比

### 4. 最近注册用户列表

表格展示最近注册的10个用户:
- ID
- 邮箱
- VIP状态 (VIP会员/普通用户)
- 计划数
- 注册时间

## 导航功能

### 顶部操作栏
- **VIP管理**: 跳转到VIP卡密管理页面
- **退出登录**: 退出管理员账号

### 页面路由
- `/admin` - 数据看板 (默认页面)
- `/admin/vip` - VIP卡密管理
- `/admin/login` - 管理员登录

## API接口

### 1. 获取统计数据
```
GET /api/v1/dashboard/stats
Headers: Authorization: Bearer {admin_token}
```

返回数据:
```json
{
  "users": {
    "total": 9,
    "vip_count": 2,
    "vip_ratio": 22.22,
    "recent_7days": 9,
    "active_30days": 5
  },
  "plans": {
    "total": 5,
    "completed": 1,
    "active": 0,
    "paused": 0,
    "draft": 1,
    "completion_rate": 20.0,
    "recent_7days": 5
  },
  "cards": {
    "total": 4,
    "used": 2,
    "unused": 2,
    "usage_rate": 50.0
  },
  "difficulty": {
    "beginner": 5
  }
}
```

### 2. 获取用户列表
```
GET /api/v1/dashboard/users?limit=10
Headers: Authorization: Bearer {admin_token}
```

返回数据:
```json
[
  {
    "id": 9,
    "email": "user@example.com",
    "is_vip": false,
    "vip_expire": null,
    "plan_count": 0,
    "created_at": "2026-03-03T03:37:22"
  }
]
```
