## 核心限制策略

### 1. 学习计划数量限制 ⭐⭐⭐⭐⭐

**免费版**: 最多3个学习计划
**VIP版**: 无限制

**实现**:
- 创建计划前检查数量
- 达到限制时提示升级VIP
- VIP用户无限制创建

**吸引力**:
- 用户通常有多个学习目标
- 3个计划很容易用完
- 强烈的升级动机

### 2. AI使用次数限制 ⭐⭐⭐⭐⭐

**免费版**: 每天3次AI调用

**实现**:

- VIP用户无限制使用

**吸引力**:
- AI是核心功能

- 高价值功能

## 技术实现

### 后端

**限制检查服务** (`vip_limits.py`):
```python
- check_plan_limit()      # 检查计划数量
- check_ai_usage_limit()  # 检查AI使用次数
- get_user_limits()       # 获取用户限制信息
```

**API端点**:
```
GET /api/v1/vip/limits    # 获取用户限制信息
POST /api/v1/plans        # 创建计划(检查限制)
POST /api/v1/analysis/generate  # AI分析(检查限制)
```
