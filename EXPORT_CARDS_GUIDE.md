# VIP卡密导出功能说明

## 功能概述

管理员可以根据条件筛选并导出VIP卡密为CSV文件,方便批量管理和分发。

## 功能特性

### 1. 筛选条件

**按有效天数筛选**:
- 全部天数
- 7天
- 30天
- 90天
- 180天
- 365天

**按使用状态筛选**:
- 全部状态
- 未使用
- 已使用

### 2. 导出内容

CSV文件包含以下字段:
- 卡密 (card_code)
- 有效天数 (duration_days)
- 状态 (已使用/未使用)
- 使用者ID (used_by)
- 使用时间 (used_at)
- 创建时间 (created_at)

### 3. 文件命名

格式: `VIP卡密_{天数}_{状态}_{日期}.csv`

示例:
- `VIP卡密_30天_未使用_2026-03-03.csv`
- `VIP卡密_全部_已使用_2026-03-03.csv`
- `VIP卡密_90天_全部_2026-03-03.csv`

## 使用场景

### 场景1: 导出未使用的30天卡密
1. 点击"导出卡密"按钮
2. 选择"有效天数: 30天"
3. 选择"使用状态: 未使用"
4. 点击"确认导出"
5. 获得CSV文件,可批量分发给用户

### 场景2: 导出所有已使用的卡密
1. 点击"导出卡密"按钮
2. 选择"有效天数: 全部天数"
3. 选择"使用状态: 已使用"
4. 点击"确认导出"
5. 获得CSV文件,用于统计分析

### 场景3: 导出所有卡密
1. 点击"导出卡密"按钮
2. 选择"有效天数: 全部天数"
3. 选择"使用状态: 全部状态"
4. 点击"确认导出"
5. 获得完整的卡密数据库

## 技术实现

### 前端实现

**导出逻辑**:
```typescript
1. 从API获取所有卡密数据
2. 根据筛选条件过滤数据
3. 生成CSV格式内容
4. 添加BOM支持中文
5. 创建Blob并触发下载
```

**CSV格式**:
```csv
卡密,有效天数,状态,使用者ID,使用时间,创建时间
1NX560PXUZGP9RGP,30,已使用,6,2026-03-03 11:31:27,2026-03-03 03:29:55
FSTK1N3F2ACTPAJU,30,未使用,,,2026-03-03 03:29:55
```

### 特性

**中文支持**:
- 添加UTF-8 BOM标记
- 确保Excel正确显示中文

**文件下载**:
- 使用Blob API
- 自动触发浏览器下载
- 清理临时URL
